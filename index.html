Security & privacy
Your code stays yours.
Where does my code run?
In an ephemeral container created for each review. The container is destroyed after the review completes. We don't have a persistent copy of your codebase.
What is stored?
Only the review output (verdict, metrics, annotations). Your source code is never stored, logged, or persisted.
Is my code sent to external LLMs?
The diff (not the full codebase) is sent to LLM providers (Anthropic, OpenAI, Google) for review analysis. Only the changed code is transmitted. If this is a blocker, we can discuss alternatives during setup.
What we don't promise yet
We're in private beta. We don't yet have SOC2, GDPR certification, or on-prem deployment. We're honest about our stage. If you need enterprise compliance, we're not ready — but we're building toward it.
